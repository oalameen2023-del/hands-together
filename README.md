# Hands Together - Non-Profit Landing Page

A clean, modern, and responsive landing page for a non-profit organization. Built with pure HTML and CSS for easy deployment and maintenance.

## Features

- Responsive design that works on all devices
- Modern and clean user interface
- Fast loading and optimized for performance
- No JavaScript dependencies
- Easy to customize and maintain

## Deployment

This website can be easily deployed to various platforms. Here are the instructions for popular hosting services:

### Deploying to Netlify

1. Create a Netlify account at [netlify.com](https://www.netlify.com)
2. Click on "New site from Git"
3. Connect your repository
4. Select the branch you want to deploy
5. Click "Deploy site"

### Deploying to Vercel

1. Create a Vercel account at [vercel.com](https://vercel.com)
2. Install Vercel CLI: `npm i -g vercel`
3. Run `vercel` in the project directory
4. Follow the prompts to deploy

## Customization

To customize the website:

1. Edit the content in `index.html`
2. Modify the styles in `styles.css`
3. Replace the hero image URL in `styles.css` with your own image
4. Update colors by modifying the color values in `styles.css`

## Structure

- `index.html` - Main HTML file containing the website structure
- `styles.css` - CSS styles for the website
- `README.md` - Documentation and deployment instructions
